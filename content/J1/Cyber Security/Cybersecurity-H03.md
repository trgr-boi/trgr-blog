## 3.1 Bedreigingen en aanvallen
- Interne vs. externe aanvallen:
    - Interne aanvallen komen van binnen de organisatie (medewerkers, contractpartners)
        - Kunnen grotere schade aanrichten door directe toegang
        - Hebben kennis van het netwerk en systemen
    - Externe aanvallen komen van buitenaf
        - Maken gebruik van kwetsbaarheden
        - Gebruiken vaak social engineering

---
### Opkomst van mobiele apparaten en IoT
- BYOD (Bring Your Own Device) vormt nieuwe uitdagingen
    - Moeilijk centraal te beheren en updaten
    - Vormt groeiende bedreiging
- Internet of Things (IoT) brengt nieuwe risico's
    - Miljarden verbonden apparaten
    - Beveiliging vaak ondermaats
    - Vormt aantrekkelijk doelwit voor aanvallers

---
## 3.2 Malware en kwaadaardige code
- Verschillende types malware:
    - Virussen: code die zich hecht aan uitvoerbare bestanden
    - Wormen: zelf-replicerende malware die via netwerk verspreidt
    - Trojaanse paarden: malware verborgen in legitieme software
    - Logic bombs: malware die activeert op specifiek moment
    - Ransomware: versleutelt bestanden voor losgeld
    - Backdoors/Rootkits: creëren verborgen toegang tot systeem
    - Keyloggers: registreren toetsaanslagen

---
## 3.3 Misleiding en oplichting
- Social engineering technieken:
    - Phishing: misleidende e-mails/websites voor gegevensdiefstal
    - Pretexting: misleiding via verhaal/voorwendsel
    - Vishing: telefonische oplichting
    - Smishing: SMS-oplichting
    - Quishing: QR-code oplichting
    - Cat phishing: misleiding via valse online identiteit
    - Spear phishing/whaling: gerichte phishing op specifieke personen
    - Shoulder surfing: meekijken bij invoeren gegevens
    - Dumpster diving: informatie zoeken in afval

---
## 3.4 Netwerkaanvallen
- Belangrijke netwerkaanvallen:
    - Botnets: netwerk van geïnfecteerde computers
    - DDoS: overbelasting van diensten
    - Sniffing: afluisteren netwerkverkeer
    - Spoofing: vervalsen van identiteit/gegevens
    - Man-in-the-middle: onderscheppen communicatie
    - Rogue Access Points: valse toegangspunten

---
## 3.5 Applicatie-aanvallen
- Types applicatie-aanvallen:
    - Zero-day: gebruik van onbekende kwetsbaarheden
    - Cross-site scripting (XSS): injecteren van scripts
    - Code injections: injecteren van kwaadaardige code
    - Buffer overflow: overschrijden van geheugenlimiet
    - Remote Code Execution: uitvoeren code op afstand

---
## 3.6 APT's (Advanced Persistent Threats)
- Kenmerken:
    - Advanced: geavanceerde middelen en technieken
    - Persistent: langdurige aanwezigheid
    - Threat: gericht op diefstal/spionage
- Voorbeelden:
    - Stuxnet: aanval op Iraanse nucleaire faciliteiten
    - Belgacom-hack: spionage door buitenlandse inlichtingendiensten
